📊 Analyse des Campagnes Marketing & Comportement Client
🧠 Contexte du projet

Dans le cadre de ce projet, j'interviens en tant que Data Analyst Junior afin d'analyser l'efficacité des campagnes marketing et d'identifier les produits les plus achetés par les clients.

L'entreprise souhaite mieux comprendre :

L'impact des campagnes sur les ventes
Le comportement d'achat des clients
Les produits les plus performants

Le jeu de données utilisé contient des informations sur :

Les clients (données démographiques)
Les transactions
Les campagnes marketing
Les produits achetés
🎯 Objectifs
Réaliser une Analyse Exploratoire des Données (EDA)
Effectuer des tests statistiques pour valider des hypothèses
Transformer les données avec Power Query
Modéliser les données dans Power BI
Construire un tableau de bord interactif orienté business
🛠️ Outils et technologies
Python : pandas, numpy, seaborn, matplotlib
Statistiques : t-test, ANOVA, test Chi², corrélation de Pearson
Power BI : modélisation + tableau de bord
Power Query : transformation des données
GitHub : gestion du projet
📌 Étapes du projet
🔍 1. Analyse Exploratoire des Données (EDA)
Chargement des données avec pandas
Identification des types de variables :
Les variables continuent (revenus, dépenses…)
Variables discrètes (nombre d'achats…)
Statistiques descriptives :
Type Moyenne, Médiane, Min, Max, Écart
📊 Visualisations :
Histogrammes → distribution des ventes
Boxplots → détection des valeurs aberrantes
Graphiques à barres → campagnes acceptées
🧹 Nettoyage des données :
Gestion des valeurs manquantes
Suppression des doublons
Détection des anomalies
💡 Perspectives :
Identification des profils clients
Premières tendances sur les campagnes
📈 2. Statistiques des tests
Test t / ANOVA :
Comparaison des ventes selon les campagnes
Test Chi² :
Relation entre variables catégorielles (ex : pays vs campagne)
Corrélation de Pearson :
Lien entre variables démographiques et comportement d'achat
🎯 Interprétation :
Analyse des valeurs p
Identification des facteurs significatifs
🔄 3. Transformation des Données (Power Query)
Pivot / Unpivot des données :
Campagnes → colonne unique
Produits → colonne unique
Plateformes → Web / Boutique / Catalogue
Nettoyage :
Renommage des colonnes
Structuration des tables
Vérification des relations :
Clé principale :ID Client
🧩 4. Modélisation des Données (Power BI)
Création du modèle relationnel
Définition des relations entre tables
Configuration des filtres
📐 Mesures DAX :
Total des ventes
Ventes par campagne
Ventes par produit
Nombre de clients par campagne
📊 5. Tableau de bord – Performances de la campagne
📌 Visualisations clés :
Nombre de clients par campagne
Chiffre d'affaires par campagne
Ventes par produit (graphique à barres empilées)
Répartition des achats par plateforme
🎛️ Interactivité :
Trancheurs (campagne, produit, pays…)
Filtres dynamiques
🎯 Objectif :

Un tableau de bord clair, intuitif et orienté décision permettant :

D'identifier les campagnes les plus performantes
D'optimiser les stratégies marketing
